# OIBSIP
new repo
🚀 Excited to share my latest Java mini-project: **Guess The Number Game** 🎮

I built a console-based game in Java where the computer generates a random number and the player has to guess it within a limited number of attempts.

✨ Features:
• Random number generation (1–100)
• Hint system (Too High / Too Low)
• Limited attempts per round
• Multiple rounds support
• Score calculation based on attempts remaining
• Input validation for a better user experience
